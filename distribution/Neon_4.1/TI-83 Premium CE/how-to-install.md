To install Neon, you have to send the following files to your calculator :
- NEON.0.8xv
- NEON.1.8xv
- NEON.2.8xv
- NEON.3.8xv
- NEONINST.8xp

You can also send LAUNCHER.8xp if you want to use a graphical menu for launching Neon programs.

Then, run NEONINSTALL.8xp to install Neon as an app. You can then remove all those files (except LAUNCHER.8xv)
